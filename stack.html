    L'Envers du Décor

    Ici, pas de constructeur de page automatique ou de solution toute faite. J'ai choisi de coder ce site à la main pour garder le contrôle total sur mon projet.

    Découvrez comment ce site a été créé. Nowone vous présente les outils et technologies utilisés pour bâtir cette plateforme de recommandation.

    Le Frontend (Le Visuel)

    • HTML5 : Pour une structure sémantique propre.
    • CSS3 : Pour le design, le mode sombre et les accents orange.
    • JavaScript : Pour rendre le site dynamique (menu et catalogues).

    Le Workflow (Les Outils)

    • VS Code : Mon éditeur de code au quotidien.
    • Git & GitHub : Pour sauvegarder mon travail et héberger le site.
    • JSON : Pour gérer mes listes de livres facilement.
